# Time Zone Converter

A modern, user-friendly web application for converting time across different time zones. Built with Nuxt.js 3 and Tailwind CSS.

## Features

- 🌍 Convert time between any time zones instantly
- 🎨 Modern, responsive design with dark mode support
- ⚡ Fast and efficient calculations using Luxon
- 📱 PWA support for offline usage
- 🔄 Real-time updates for current time
- 💾 Recent timezone history
- 🔍 SEO optimized with schema.org data
- 🌓 Automatic dark mode detection

## Tech Stack

- Nuxt.js 3
- Tailwind CSS
- VueUse
- Pinia
- Luxon
- PWA support
- TypeScript

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Generate PWA icons:
   ```bash
   npm run generate:icons
   ```
4. Start development server:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run generate` - Generate static site
- `npm run preview` - Preview production build
- `npm run generate:icons` - Generate PWA icons
